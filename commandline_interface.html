bag2dataset¶

For large ROS bagfiles best way is to convert the rosbag beforehand with the provided command line tool.

Example¶

An example to convert topic /os1_cloud_node/point within the test.bag rosbag to the same directory.

bag2dataset test.bag /os1_cloud_node/points .

bag2dataset¶

bag2dataset [OPTIONS] BAGFILE [TOPIC] [FOLDER_TO_WRITE]

Options

-s, --start <start_frame_number>¶
Default

0

-e, --end <end_frame_number>¶
--keep-zeros, --no-keep-zeros¶
Default

False

--max-size <max_size>¶
Default

100

--install-completion <install_completion>¶

Install completion for the specified shell.

Options

bash | zsh | fish | powershell | pwsh

--show-completion <show_completion>¶

Show completion for the specified shell, to copy it or customize the installation.

Options
